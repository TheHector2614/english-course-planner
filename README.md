# English Course Planner

Plan de estudios interactivo de inglés desde A1 hasta B2+.
Diseñado con habilidades de diseño UI profesionales (ui-skills.com).

## Contenido

- **plan-A1-B2-completo.html**: Plan de estudios completo con quizzes, flashcards,
  ejercicios de escritura, referencias a PDF, guía para el profesor.
- **ENGLISH/**: Biblioteca de 23 recursos PDF (gramática, vocabulario, cursos).
- **opencode.json**: Configuración para OpenCode AI agents.

## Niveles

| Nivel | Horas | Enfoque |
|-------|-------|---------|
| A1 | 8h | Fundamentos |
| A2 | 10h | Construcción |
| B1 | 12h | Expansión |
| B1+ | 12h | Consolidación |
| B2 | 14h | Dominio |
| B2+ | 14h | Fluidez profesional |

## Cómo usar

Abrir `plan-A1-B2-completo.html` en cualquier navegador.
No requiere conexión a internet (excepto Google Fonts al cargar).
