UNIT 1: GETTING ACQUAINTED (Familiarizándose / Conociéndose)

  - Conocer a alguien nuevo.
  - Identificar y describir personas.
  - Proporcionar información personal.
  - Presentar a alguien a un grupo.
* Vocabulary (Vocabulario):
  - Títulos formales.
  - Adjetivos positivos para describir personas.
  - Información personal.
  - Países y nacionalidades.
* Grammar (Gramática):
  - Preguntas de información con el verbo "be": Revisión.
  - Contracciones.
  - Modificación con adjetivos: Revisión.
  - Adjetivos positivos.
  - Preguntas de sí/no y respuestas cortas con el verbo "be": Revisión.

  - Comenzar las respuestas con una pregunta para confirmar.
  - Usar "Let's" para sugerir un curso de acción.

* Listening Skills / Pronunciation (Escucha y Pronunciación):
  - Escuchar para identificar detalles.
  - Inferir información.
  - Pronunciación: Entonación de las preguntas.

UNIT 4: FOOD AND RESTAURANTS (Comida y Restaurantes)

* Communication Goals (Metas de comunicación):
  - Pedir una recomendación de restaurante.
  - Ordenar desde un menú.
  - Hablar con un mesero o mesera y pagar la cuenta.
  - Discutir sobre comida y salud.
* Vocabulary (Vocabulario):

  - Categorías de alimentos.
  - Grados de hambre.

  - Clarificar una petición pidiendo información más específica.
  - Indicar un pensamiento repentino con "Actually".
  - Usar "I'll have..." para ordenar a un mesero.
  - Incrementar la cortesía usando "please".
* Listening Skills / Pronunciation (Escucha y Pronunciación):
  - Escuchar para tomar notas y predecir.
  - Inferir la ubicación de una conversación.

* Reading / Writing (Lectura y Escritura):
  - Textos: Menús, un sitio web de nutrición y una historia en fotos.
  - Habilidades: Interpretar un mapa, comprender desde el contexto e inferir información.
  - Tarea escrita: Escribir un artículo corto sobre comida para un blog de viajes (Enfoque: Conectar palabras y frases con "and" o "in addition").
